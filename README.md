# NITR_MarketZone

**NITR_MarketZone** is an online marketplace specifically designed for students of NIT Rourkela. It allows students to list items they want to sell, and other students can browse, negotiate prices, and purchase these items. The platform also features a chat system for students to communicate about the items they're interested in, making the process more interactive and user-friendly. The application includes a secure login system where students can sign up using their college email, and a verification link is sent to their college email for account verification. Additionally, the platform offers various filters like price, item type, discount, and the latest items to help users find what they need more efficiently.

## Features

- **Login & Verification**:  
  Students can log in using their college email. A verification link will be sent to their college email for account validation.

- **Buy & Sell Items**:  
  Students can list the items they want to sell, including a description and price. Other students can browse and buy these items.

- **Price Negotiation**:  
  Buyers and sellers can negotiate prices directly through the platform.

- **Chat System**:  
  Integrated chat functionality allows students to discuss the items they are interested in directly on the platform.

- **Filters**:  
  - **Price Filter**: Users can filter items based on their price range.
  - **Item Type Filter**: Allows users to sort items by their type (e.g., electronics, furniture, books, etc.).
  - **Discount Filter**: Users can find items that are available at discounted prices.
  - **Latest Items**: Users can sort items by the most recently listed items.

- **User-friendly Interface**:  
  Designed with simplicity and usability in mind, the interface is easy to navigate for students of all technical levels.
